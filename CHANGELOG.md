# Changelog

## 0.1.0 — 2026-03-22

Initial release of tokenBar — a focused fork of [CodexBar](https://github.com/steipete/CodexBar).

### What's different
- Stripped down to core providers: **Claude**, **Codex**, **Gemini**, **Antigravity**
- Removed 20+ unsupported provider integrations
- Removed CLI, Widget, and helper modules
- Full rebrand: CodexBar → tokenBar
- New app icon
- Simplified codebase and reduced dependencies
